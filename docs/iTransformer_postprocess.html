xLSTMStack 细化架构

配置示例：rnn_type=sLSTM，layers=2，share_params=True，enable_longconv=True，kernels={3,7,15,25}，hidden=H。

说明： 1) 每层次序为 LayerNorm → LongConvMix → RNN → 残差； 2) LongConvMix 采用深度可分多核空洞/普通卷积并以 1x1Conv 聚合，输出经门控 σ(g) 与输入相加； 3) RNN 支持 sLSTM/GRU，当前模型为 sLSTM； 4) share_params=True 时，RNN 与 LongConvMix 在多层间复用权重（每层仍有独立 LayerNorm）； 5) 残差连接分别在 LongConvMix 与 RNN 之后，有助于梯度稳定与多尺度融合。

LongConvMix 细化架构

多核深度可分时序卷积（groups=H），保持长度不变（padding=k//2），门控融合并残差回加。
flowchart LR X["输入 h (B, L, H)"] T1["transpose → (B, H, L)"] subgraph DW["Depthwise Conv 分支（m 个 kernel）"] D3["DWConv k=3 → (B, H, L)"] D7["DWConv k=7 → (B, H, L)"] D15["DWConv k=15 → (B, H, L)"] D25["DWConv k=25 → (B, H, L)"] end CAT["通道维 concat → (B, H×m, L)"] P1["Pointwise 1×1Conv → (B, H, L)"] DO["Dropout(p)"] T2["transpose → (B, L, H)"] G["可学习门 g ∈ ℝ → σ(g)"] OUT["残差融合: y = x + σ(g)·mix → (B, L, H)"] X --> T1 T1 --> D3 T1 --> D7 T1 --> D15 T1 --> D25 D3 --> CAT D7 --> CAT D15 --> CAT D25 --> CAT CAT --> P1 --> DO --> T2 --> OUT X -. 残差 .-> OUT G --> OUT

细节： 1) Depthwise：每个卷积分组数=H，仅在时间维 L 上卷积，通道间无混合； 2) 长度保持：padding=k//2，输出与输入同 L； 3) 多尺度：kernels 由 configs.xlstm_kernels 指定（如 {3,7,15,25}），并行分支后在通道维拼接； 4) 1×1Conv：将拼接后的 (H×m) 压回 H，完成通道混合； 5) 门控：全局标量门 g（nn.Parameter）经 σ 压缩到 (0,1)，控制混入强度； 6) 残差：与输入 x 相加，形成可学习的前置时域混合，便于梯度流动与稳态训练； 7) 开关：若 enable_longconv=False，则 LongConvMix 退化为 Identity。
