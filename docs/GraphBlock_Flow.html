GraphBlock 图卷积模块详细流程

基于仓库 utils/MSGBlock.py 的实现，纯 HTML + SVG 可视化。包含：邻接构建 → mixprop 多跳传播 → 头尾卷积与残差归一化。

阅读指引

  • 左列：特征流（输入 → start_conv → mixprop → end_conv → 线性 → 残差 + LN → 输出）。
  • 右列：邻接矩阵构建（自适应 + 时间门控偏置 → Softmax → 加自环 → 行归一化）。
  • 中部：mixprop 传播细节（gdep 次迭代，α 控制自信息与邻域信息权衡）。

要点总结

  1. 邻接矩阵 = 自适应节点嵌入产出的 logits + 时间门控偏置，经行 Softmax 得到 A；随后加自环并行归一化得到 Â。
