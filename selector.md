# 可微特征选择器构建指南

## 1. 概述

可微特征选择器是一种能够在神经网络训练过程中自动学习特征重要性的机制。它通过引入可学习的掩码参数，将离散的特征选择过程转化为连续的、可微的运算，从而实现端到端的特征选择。

## 2. 核心原理

### 2.1 基本概念
- **可微性**：整个选择过程必须是连续且可导的，以便通过梯度下降进行优化
- **掩码机制**：使用可学习的参数生成一个0-1之间的掩码，与输入特征相乘
- **稀疏性约束**：通过正则化项鼓励掩码趋向于0或1，实现特征选择

### 2.2 数学表示
给定输入特征 $x = [x_1, x_2, ..., x_n]$，引入可学习参数 $z = [z_1, z_2, ..., z_n]$，通过Sigmoid函数生成掩码：
$$m = \sigma(z) = [\sigma(z_1), \sigma(z_2), ..., \sigma(z_n)]$$
掩码后的特征为：
$$x' = x \odot m = [x_1 \cdot m_1, x_2 \cdot m_2, ..., x_n \cdot m_n]$$

### 3.2 可微特征选择层
- **参数初始化**：使用较小值初始化掩码参数，使初始掩码接近0.5
- **掩码生成**：使用Sigmoid或Gumbel-Softmax函数生成连续掩码
- **训练/推理差异**：训练时使用软掩码，推理时可根据需要转换为硬掩码

### 4.1 主要组成部分

### 4.2 任务损失
- 回归任务：均方误差(MSE)
- 分类任务：交叉熵损失(Cross-Entropy)

### 4.3 稀疏性正则化
- **L1正则化**：$L_{reg} = \sum_{i=1}^n |m_i|$
- **熵正则化**：鼓励掩码接近0或1，减少不确定性
- **预算约束**：限制被选择特征的数量不超过指定值

## 5. 训练策略

### 5.1 训练流程
1. 前向传播：计算掩码并应用于输入特征
2. 计算损失：结合任务损失和正则化项
3. 反向传播：更新主干网络参数和掩码参数
4. 迭代优化：重复上述过程直至收敛

### 5.2 超参数设置
- **正则化强度λ**：控制特征选择的稀疏程度
- **学习率**：掩码参数的学习率可略高于主干网络
- **温度参数**（如使用Gumbel-Softmax）：控制近似程度
